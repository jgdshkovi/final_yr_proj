Features & Usage


  • The Website is responsive & one can select from dr0p-d0wn either:

    • - to check the Classifier with a Sample toxic text to find its toxicity or

    - to enter Twitter UserName to get Toxic Tweets from the timeline

  • For the sample text user tests, we simply return the Toxicity score

  • But for getting Toxic tweets using UserName (if any), one can select a threshold using the slider & only the tweets above the toxicity threshold score are returned



    • Scope


  • We were only able to find the toxicty for the tweets, as twitter provides limited access to them using Tweepy API. But this can be further improved if we were to scrape the replies for the tweets (using tools like selenium), as most toxic stuff is found discussions, and working on them.

  • Also, we can easily get data from various Social Media Platforms(SMPS), using APIs or some scraping tools & integrate our Toxic Comment Idenfifier onto them.

  • The Source Code is available on GitHub, so anyone of 'U' can make a use use of it or contribute to the project.
